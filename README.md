```bash
python3 -m http.server 5173
```


## Загрузка на GitHub + Vercel

1. Создать новый GitHub repository, например `sustav-plus-miniapp`.
2. Загрузить все файлы из папки проекта в корень репозитория.
3. Открыть Vercel.
4. Import Project → выбрать GitHub repository.
5. Framework Preset: Other / Static.
6. Build Command: оставить пустым.
7. Output Directory: оставить пустым или `.`.
8. Deploy.
9. Получить HTTPS-ссылку вида:

```text
https://sustav-plus-miniapp.vercel.app
```

## Подключение в Telegram

1. Открыть `@BotFather`.
2. `/mybots`.
3. Выбрать бота.
4. Bot Settings → Menu Button / Configure Mini App.
5. Вставить HTTPS-ссылку Vercel.
6. Название кнопки: `Открыть Сустав+`.

## Важно по медицине

Тексты внутри приложения — образовательные чек-листы и навигация. Они не заменяют врача, хирурга, реабилитолога или экстренную медицинскую помощь. В реальном запуске нужно согласовать медицинские формулировки со специалистами и добавить юридические документы: пользовательское соглашение, политику конфиденциальности, согласие на обработку данных.
