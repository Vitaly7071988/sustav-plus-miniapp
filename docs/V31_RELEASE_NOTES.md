# Сустав+ v3.1 — исправление отображения картинок упражнений

Новые 4-шаговые карточки упражнений вертикального формата больше не обрезаются в деталке упражнения.

Изменения:

- CSS переопределяет `.exercise-photo`: `height:auto`, `aspect-ratio:auto`, `object-fit:contain`;
- CSS переопределяет `.exercise-illustration`: без принудительной минимальной высоты и без обрезания содержимого;
- обновлён query-string изображений на `?v=31-fullimg`, чтобы Telegram/Vercel не держали старый кэш;
- сохранены серверные файлы бота `/api/bot.js`.
