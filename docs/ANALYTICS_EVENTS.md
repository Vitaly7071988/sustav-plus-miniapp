# События аналитики Сустав+

Основные события:

- `app_open` — открытие приложения
- `screen_view` — просмотр экрана
- `nav_click` — переход по кнопке навигации
- `program_open` — открытие маршрута 14/30 дней
- `day_open` — открытие конкретного дня
- `exercise_open` — открытие упражнения
- `task_toggle` — отметка задачи
- `exercise_toggle` — отметка упражнения
- `routine_toggle` — отметка рутины
- `water_change` — изменение воды
- `tracker_save` — сохранение трекера
- `assistant_ask` — вопрос AI/помощнику
- `profile_save` — сохранение профиля
- `team_save` — сохранение команды
- `question_toggle` — отметка вопроса врачу
- `external_click` — клик по внешней ссылке

Что смотреть в первую очередь:

1. `screen_view` по экранам:
   - nutrition
   - weight-loss-nutrition
   - preop
   - exercises
   - concierge
   - longlife
2. `external_click`:
   - victoria_weight_loss
   - victoria_nutrition_direct
   - concierge_contact_vitaly
3. `exercise_open`
4. `tracker_save`

Это покажет не только сколько людей вошли, но и что им реально интересно.
