# Сустав+ Mini App v2.4 — нативный цифровой проводник

## Что добавлено

- Новый верхний экран «Сегодня ваш шаг».
- Мягкая логика: страх → этап → первый шаг → действие → прогресс → помощь Виталия.
- Разделы v2.4: Сегодня, Маршрут, Разминка, Вопросы, Трекер, Помощь.
- Взрослая геймификация без инфантильности: задачи дня, прогресс, уровень, серия.
- Лёгкая разминка: 3 минуты в кровати, 5 минут у опоры, разогнать кровь, я боюсь двигаться, офис/дом.
- Расширенный раздел вопросов: операция, врач, страх, дом, первые дни, ЛФК, красные флаги, близкие.
- Нативная продажа личного разбора 4 900 ₽.
- Ссылки на Google-трекер, WhatsApp Виталия и Telegram.
- События аналитики отправляются в `/api/track` с `appVersion: 2.4-native-guide-game`.

## Как обновить

1. Распаковать архив.
2. Залить все файлы поверх текущего репозитория GitHub.
3. Commit changes.
4. Дождаться деплоя Vercel.
5. При кэше Telegram обновить URL в BotFather с параметром `?v=24`.

## Важно

Сустав+ не заменяет врача, хирурга или реабилитолога. Это цифровой проводник, который помогает снять хаос, собрать вопросы и понять следующий шаг.
