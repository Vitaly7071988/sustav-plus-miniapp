# Сустав+ v2.8 — визуалы упражнений в фирменном стиле

Что обновлено:

- Полностью заменены изображения упражнений в `assets/exercises/`.
- Все карточки сделаны в едином стиле Сустав+: тёплый фон, пыльная роза, оливковый, S+.
- Каждая инструкция теперь показывает 4 понятных шага.
- Убраны рискованные позы, йога-подобные положения и визуальные ошибки с лишними конечностями.
- Для стоячих действий добавлена опора/костыли там, где это нужно по смыслу.
- Лестница, ходьба, стул, кровать, машина, туалет, душ оформлены как бытовые безопасные навыки, а не как спорт.

Важно: материалы не заменяют врача или реабилитолога. Пользователь выполняет только то, что разрешено специалистом.
