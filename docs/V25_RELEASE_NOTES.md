# Сустав+ Mini App v2.5 — fixed navigation + clearer route flow

## Что исправлено

- Убрано наложение нового блока поверх старого приложения.
- Mini App теперь рендерится как единое standalone-приложение через `v25-native-guide.js`.
- Нижнее меню теперь рабочее: Главная / Маршрут / Разминка / Трекер / Вопросы / Помощь.
- Кнопки на первом экране теперь рабочие.
- Раздел «Где вы сейчас?» теперь работает правильно: после выбора этапа меню выбора уходит, пользователь попадает сразу в свой маршрут.
- Выбор этапа сохраняется в `localStorage`.
- События пишутся в `/api/track` с `appVersion: 2.5-native-guide-fixed-nav`.

## Что обновлено в визуалах упражнений

- Обновлены основные картинки упражнений в `assets/exercises/`.
- Основные карточки сделаны в формате 4 шагов.
- Добавлены более понятные изображения для взрослой аудитории: без поз йоги, с опорой/костылями там, где это нужно.
- Обновлены: дыхание, стопы, бедро, ягодицы, скольжение пяткой, отведение ноги, вставание/посадка, ходьба с 2 локтевыми костылями, лестница с 2 локтевыми костылями.

## Важно

Материалы приложения не заменяют врача, хирурга или реабилитолога. При красных флагах пользователь должен обращаться за медицинской помощью.
