# Сустав+ v0.4 — release notes

## Цель версии

Сделать приложение похожим не на красивую памятку, а на первую серьёзную систему сопровождения пациента.

## Новые экраны

1. **Старт / онбординг**
   - объясняет продукт;
   - фиксирует, что приложение не заменяет врача;
   - переводит пользователя в анкету.

2. **Анкета**
   - сустав;
   - этап;
   - дни после операции;
   - тип опоры;
   - ограничения врача.

3. **Безопасность**
   - светофор боли;
   - красные флаги;
   - напоминание, когда не ждать ответа приложения.

4. **Можно / нельзя**
   - дни 1–7;
   - дни 8–14;
   - дни 15–30.

5. **Вопросы врачу**
   - чек-лист для консультации;
   - можно отмечать заданные вопросы.

6. **Моя команда**
   - хирург;
   - реабилитолог;
   - клиника;
   - дата операции;
   - следующий осмотр.

7. **Спокойствие**
   - психологическая поддержка;
   - мини-практика дыхания;
   - нормализация страхов и откатов.

8. **Видео от Виталия**
   - заглушка под приветственное видео 30–45 секунд.

9. **Итог недели**
   - количество записей;
   - средняя боль;
   - мин/макс боль.

## Обновления существующих экранов

- Главный экран: добавлены быстрые кнопки «Первый вход», «Видео», «Вопросы врачу».
- Профиль: добавлены ссылки на безопасность, вопросы врачу, команду, можно/нельзя, итог недели.
- Маршрут: добавлен блок «Светофор боли» прямо внутри дня.
- Трекер: добавлены сон и ходьба.
