# Сустав+ Mini App v2.9 — полное объединение v2.3 + новые улучшения

## Главное

Эта версия возвращает полную продуктовую базу v2.3 и добавляет актуальные улучшения v2.4–v2.8 без потери старых разделов.

## Что сохранено из v2.3

- системная кнопка «← Назад»;
- маршруты 14/30 дней;
- анкета и профиль/«Мой путь»;
- трекер;
- питание;
- раздел для близких;
- комбо-наборы;
- Google Sheets аналитика;
- AI-заготовка;
- подготовка дома;
- красные флаги;
- вопросы врачу;
- контакты и экосистема.

## Что добавлено

- версия `APP_VERSION = 2.9-full-merge-exercises-bot`;
- новая расширенная база из 25 упражнений;
- новые 4-шаговые картинки упражнений в стиле Сустав+;
- подъём на носочки у опоры;
- перенос веса у опоры;
- шаги на месте у опоры;
- бытовые навыки: кровать, машина, туалет, душ;
- обновлённая лестница с двумя костылями;
- сохранён `api/bot.js` для ответа на `/start`.

## Важно по Telegram-боту

Токен не нужно создавать заново при каждой версии.

Один раз в Vercel должна быть переменная:

```text
TELEGRAM_BOT_TOKEN=ваш_токен_из_BotFather
```

После загрузки новой версии:

1. Сделать Redeploy в Vercel.
2. При необходимости заново открыть:

```text
https://api.telegram.org/botВАШ_ТОКЕН/setWebhook?url=https://sustav-plus-miniapp.vercel.app/api/bot
```

3. Проверить `/start` в `@sustavplus_app_bot`.
