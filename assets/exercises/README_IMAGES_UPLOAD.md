# Сустав+ — обновление картинок упражнений

Папка `assets/exercises/` внутри этого архива содержит новые PNG-картинки для замены в Mini App.

## Как загрузить
1. Распакуйте архив.
2. На GitHub откройте репозиторий `sustav-plus-miniapp`.
3. Перейдите в папку `assets/exercises`.
4. Нажмите `Add file` → `Upload files`.
5. Перетащите PNG-файлы из папки `assets/exercises` этого архива.
6. GitHub спросит про замену файлов — это нормально.
7. Нажмите `Commit changes`.
8. Дождитесь деплоя Vercel.
9. Откройте Mini App с параметром `?v=images-01` или обновите URL в BotFather.

## Важно
Файлы `seated-knee-extension.png` и `standing-hip-extension.png` добавлены как новые картинки, но они появятся в приложении только если эти упражнения добавлены в код.
